
# 最短公共父序列

最短公共父序列 (SCS) `X`和`Y`是最短的序列`X`和`Y`作为后果. 

换句话说,假设我们给 两个字符串str1和str2,找到带有他们的最短的字符串,其中 str1和str2 都作为 子序列. 

这是与 最长递增子序列问题 密切相关的问题. 

## 例

    Input:   str1 = "geek",  str2 = "eke"
    Output: "geeke"

    Input:   str1 = "AGGTAB",  str2 = "GXTXAYB"
    Output:  "AGXGTXAYB"
