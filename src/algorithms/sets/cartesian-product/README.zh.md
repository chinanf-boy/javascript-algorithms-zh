
# 笛卡尔积

在集合论中,笛卡尔积是一种数学运算,它从 多个集合 (或 产品集和 或 简单乘积 )  中 返回集合. 也就是说,对于集合A和B,笛卡尔乘积A×B 是 所有有序对 (a,b) 的集合,其中 a∈A和b∈B. 

`A={x,y,z}`和`B={1,2,3}`的笛卡尔积`AxB`
