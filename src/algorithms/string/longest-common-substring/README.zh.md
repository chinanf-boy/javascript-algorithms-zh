
# 最长公共子串

最长公共子串问题是找到 最长的字符串 (或字符串) ,它是两个或多个字符串的子字符串 

## 例

`ABABC`,`BABCA`和`ABCBA`字符串中最长公共子串是`ABC`长度为3.其他常见的子串是`A`,`AB`,`B`,`BA`,`BC`和`C`. 

    ABABC
      |||
     BABCA
      |||
      ABCBA
