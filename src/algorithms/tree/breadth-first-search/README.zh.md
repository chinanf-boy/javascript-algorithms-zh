
# 广度优先搜索 (BFS) 

广度优先搜索 (BFS) 是用于 遍历或搜索 树或图数据结构 的算法. 它从树根 (或图的某个任意节点,有时称为"搜索节点") 开始,并在移动到下一级邻居之前,先探索邻居节点.
