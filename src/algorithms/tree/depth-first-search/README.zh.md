
# 深度优先搜索 (DFS) 

深度优先搜索 (DFS) 是用于 遍历或搜索 树或图数据结构的算法. 一个从根开始 (在图的情况下选择一些任意节点作为根) 并在回溯之前尽可能地沿着每个分支进行探索.
