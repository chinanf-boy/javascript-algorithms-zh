
# 图中的桥梁

在图论中,一个**桥**,**地峡**,**切割边缘**, 要么**切弧**是图的边,将其删除会增加了连接图组件的数量. 等效地,当且仅当它不包含在任何环中时,边是一个桥. 如果图表不包含桥梁,则称其为无桥或无峡谷.
